1新建仓库，仓库名必须为 .github.io
2npm install -g hexo
3hexo init// 本地运行hexo s
4部署到github
项目根目录下找到 _congif.yml，找到 deploy 字段并填写完整 # Deployment ## Docs: https://hexo.io/docs/deployment.html deploy: type: git repo: https://github.com/TJ-XiaJiaHao/TJ-XiaJiaHao.github.io branch: master 5npm install hexo-deployer-git --save
6hexo clean hexo deploy
7查看效果：url为仓库名
